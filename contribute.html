The Desires of Feminism

Citations

Contributions play a major part in improving this project; having multiple perspectives and experiences contribute to the data helps to create a more expansive view of feminist movements across the world. That being said, there must be some cohesion to present this data in a user-friendly way. This page will contain the Values & Accountability Metrics all contributors should hold to, as well as specific instructions on how to contribute. As such, this page will change as the project develops and its methodology shifts.

For now, if you'd like to contribute you can visit the GitHub repository page and/or contact @emucc on there. More contact info and details will be added soon.
